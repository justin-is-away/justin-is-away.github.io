How and why to type faster




As an Internet native, I find typing to be pretty fun. Working on a computer is pretty inevitable these days, anyways, so it seems like a practically useful skill (to some extent).


But it can be hard! And I get that. Still, though, I think typing ~60 wpm is a pretty easily acquirable skill with minimal time investment, but it seems like most people type at a way lower speed.
